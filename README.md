# Full Stack Todo Application

This is a simple full stack Todo application built as part of a technical assignment.
The project demonstrates basic CRUD operations and frontendâ€“backend integration.

## Tech Stack
- Backend: Node.js, Express.js
- Frontend: React.js
- API Communication: Fetch API

## Features
- Add a new todo
- View all todos
- Update todo status (Pending / Completed)
- Delete a todo

## Setup Instructions

### Backend
1. Navigate to the backend folder
2. Install dependencies:
   npm install
3. Start the server:
   node app.js

The backend runs on http://localhost:5000

### Frontend
1. Navigate to the frontend folder
2. Install dependencies:
   npm install
3. Start the React app:
   npm start

The frontend runs on http://localhost:3000

## Notes
- In-memory storage is used as allowed in the assignment.
- No deployment is required.
